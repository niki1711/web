# Ice Scoop – Frontend Web Project

Ice Scoop is a simple frontend web application for an ice cream and beverage store.

## Overview
This project demonstrates a responsive UI for browsing food categories such as ice creams, drinks, waffles, and cakes.

## Features
- Category-based product browsing  
- Basic product listing interface  
- Responsive design layout  
- Simple navigation and UI components  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Purpose
This project was developed to practice frontend design and user interface development.

## Note
This is a static frontend project and does not include backend or payment integration.
